Configurações iniciais:

Baixar repositório: git clone URL

Acessar pasta do repositório: cd pasta_repositorio

git config --global user.email "you@example.com"

git config --global user.name "Your Name"

-------

Comandos básicos:

Baixar atualizações do repositório: git pull

Mandar todas as modificações feitas para a zona intermediária: git add .

Comitar todas as modificações que estão na zona intermediária: git commit -m "mensagem de commit"

Mandar todos os commits para o repositório: git push

------

Outros comandos:

Ver o que você modificou desde o último clone/pull: git status

Mandar apenas uma modificação para a zona intermediária: git add exemplo.txt
